To visit the download page, click here

Scroll down to the download section and click on the file download according to your computer model (I have 64 bit) and make sure the version is the latest one (V1.4.3)


After the download is complete, if you see a warning sign, right click on the file and click on keep
You will then be taken to a place that has 3 options; go back, Delete or show more options. Click on show more options then click on 'Keep anyways'
After thats done, double click on the file


Click on extract all and that will generate the actual file Then open the file in the place and then you need to double click the file to open the file with contains baldi.exe, which is the game
